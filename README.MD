# PixyPix

WebApp for pixelizating images.

## Technologies

Maven

Java

JavaScript

HTML5

CSS

Servlets

Fetch

## Installation

1. install tomcat7

2. git clone https://github.com/ozahirnyi/Pixelizator

3. cd Pixelizator

4. ./mvnw tomcat7:run

## Usage

1. Choose the file

2. Input Pixel size

3 Press 'Pixelizate'
